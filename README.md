# Claude Projects

A workspace for Claude Code development and remote collaboration.

Use this repository with Claude Code on desktop or Android to sync projects across devices.

## Structure

```
├── projects/          # Individual project folders
├── scripts/           # Utility scripts
├── notes/            # Documentation and notes
└── README.md         # This file
```

## Usage with Claude Code

1. Clone this repo in Claude Code
2. Create project folders as needed
3. Push changes to sync across devices
4. Use Claude Code Android app to access remotely

## Getting Started

- Create a new folder for each project
- Use descriptive commit messages
- Keep projects organized and documented
